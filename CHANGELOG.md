- Bearer、Tx Gate、Origin Authorization、Reliable Command、Durable Eventはend-to-endで未統合です。Runtime 統合済み production durable storage も未完了です（ESP dual-slot candidate / POSIX SQLite host candidate はいずれも Runtime body 未接続; field-ready 非claim）。
- ESP-IDF port の storage FULL attestation / physical power-cut HIL、USB、Wi-Fi、SX1262、Display/Leak 実機 path は未完成です。M3-prep は component packaging、M3-basic は clock/entropy/execution adapters、control framing は private `NCG1` codec、owner/cell/loopback skeleton は [docs/22](docs/22-m3-owner-cell-agent-skeleton.md)、storage candidate は host conformance + ESP_UNPROVEN（[docs/21](docs/21-m3-esp-idf-durable-storage.md)）までです。USB/TCP transport、logical control messages、public Runtime wiring は未統合です。
- Public API、wire、storage formatの互換性はまだ保証しません。
- Security、regulatory compliance、production SLO、field readinessを保証するreleaseではありません。

このcheckpointはpre-alphaです。Fixtureやsimulator testの成功は、production deploymentや実radioの安全性・信頼性を証明しません。
