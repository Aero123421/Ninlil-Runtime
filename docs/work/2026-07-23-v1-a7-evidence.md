## handshake 段 × test

| 段 | 正常系 | 負例 | CTest 関数 | 結果 |
|---|---|---|---|---|
| request（endpoint begin） | membership ACTIVE → JOIN_REQUEST 送出 | 非メンバ / PENDING → `REJECT_MEMBERSHIP`、`success=0` | `run_handshake_success` / `run_reject_non_member` | PASS |
| challenge（controller on_request） | nonce/epoch/session 付与、CRC 検証通過 | bad CRC → decode `NINLIL_M4_LAB_CRC` | `run_handshake_success` / `run_reject_bad_challenge_crc` | PASS |
| response（endpoint on_challenge） | identity proof 生成・送出 | wrong credential → controller 側 `REJECT_CREDENTIAL`；TTL 超過 → `REJECT_EXPIRED`；同一 challenge 再処理 → `REJECT_REPLAY` | `run_handshake_success` / `run_reject_wrong_credential` / `run_reject_expired_challenge` / `run_reject_replay` | PASS |
| install（controller on_response → endpoint on_install） | install token mint、JOIN_INSTALL 送出、ATTACHED | wrong credential 時 `has_install_token=0`（mint なし） | `run_handshake_success` / `run_reject_wrong_credential` | PASS |
