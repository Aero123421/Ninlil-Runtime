## 結線（submit → delivery）

```
Runtime bearer admission (runtime_v1_capability)
  -> ninlil_c4_c5_lab_dispatch_delivery
       U6 route -> ninlil_c4_lab_usb_custody_offer -> peer custody_accept
       SIMULATED route -> ninlil_c5_lab_radio_tx_data
            -> C3 seal -> C6 transmit (R7_HOP_DATA) -> SPI sim -> peer channel
            -> ninlil_c5_lab_radio_irq_rx_admit (C3 auth/replay)
```
